ASSETS FOLDER
=============

Esta carpeta está destinada para almacenar recursos adicionales del proyecto:

- Diagramas de arquitectura
- Sketches de diseño visual
- Notas de desarrollo
- Referencias visuales
- Capturas de pantalla del juego
- Documentación adicional

Nota: El Hack Computer no soporta imágenes externas, pero esta carpeta
puede usarse para documentación y referencia durante el desarrollo.
